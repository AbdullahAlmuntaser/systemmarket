# Viso ERP Blueprint

## نظرة عامة على المشروع

Viso ERP هو نظام تخطيط موارد مؤسسات (ERP) شامل مصمم خصيصًا لشركات البيع بالتجزئة الصغيرة والمتوسطة. يهدف النظام إلى توفير حل متكامل يغطي جميع جوانب العمليات اليومية، من نقاط البيع والمحاسبة إلى إدارة المخزون والموارد البشرية.

## الميزات الرئيسية

### أ. المحاسبة ونقاط البيع (POS)

- [x] **محرك محاسبة مزدوج القيد:** ترحيل جميع المعاملات (مبيعات, مشتريات, مصروفات) إلى دفتر الأستاذ العام تلقائيًا.
- [x] **شجرة الحسابات (COA):** شجرة حسابات قياسية وقابلة للتخصيص.
- [x] **إدارة الضرائب:** دعم ضريبة القيمة المضافة (VAT) على المبيعات والمشتريات، مع تقارير ضريبية مفصلة.
- [x] **إدارة النقدية والمبيعات الآجلة:** تتبع دقيق للمعاملات النقدية والآجلة.
- [x] **وحدة نقاط بيع متكاملة (POS):** واجهة سهلة الاستخدام لتسجيل المبيعات بسرعة وكفاءة.
- [x] **إدارة الدفعات:** تسجيل دفعات العملاء والموردين وتحديث الأرصدة تلقائيًا.
- [x] **إدارة المصروفات:** تسجيل وتصنيف المصروفات التشغيلية.
- [x] **مردودات المبيعات والمشتريات:** معالجة عمليات الإرجاع وتأثيرها المحاسبي والمخزني.
- [x] **تقارير مالية:**
    - [x] **ميزان المراجعة**
    - [x] **قائمة الدخل**
    - [x] **الميزانية العمومية**
    - [x] **قائمة التدفقات النقدية**

### ب. العملاء والموردين

- [x] **إدارة علاقات العملاء (CRM):** سجل كامل للعملاء، تتبع أرصدتهم، وتاريخ معاملاتهم.
- [x] **إدارة الموردين:** سجل للموردين وتتبع الأرصدة المستحقة لهم.

### ج. إدارة المخزون

- [x] **جرد المنتجات:** تتبع دقيق لكميات المنتجات وتكلفتها.
- [x] **تتبع الدفعات وتواريخ الانتهاء:** إدارة المنتجات حسب رقم الدفعة وتاريخ الانتهاء (FIFO).
- [x] **تعدد المستودعات:** دعم العمل في عدة مواقع وفروع.
- [x] **التحويل المخزني:** ميزة نقل البضائع بين المستودعات بضوابط دقيقة.
- [x] **تحسين الفئات:** واجهة عصرية لتصنيف المنتجات.

### د. الموارد البشرية والرواتب

- [x] **سجل الموظفين:** إدارة بيانات الموظفين, المسميات الوظيفية, والرواتب الأساسية.
- [x] **مسير الرواتب:** توليد مسيرات الرواتب الشهرية آلياً مع دعم البدلات والخصومات.

### هـ. تحسينات واجهة المستخدم وتجربة المستخدم (UI/UX)

- [x] **اتساق المظهر:** تم تحديث واجهات المبيعات, المشتريات, الفئات, والعملاء لاستخدام `Theme.of(context).colorScheme` لتحسين الاتساق البصري ودعم الوضع الداكن/الفاتح.
- [x] **إعادة هيكلة المكونات:** تم استخراج منطق واجهة المستخدم المعقد إلى مكونات قابلة لإعادة الاستخدام, مثل `SaleDetailsBottomSheet` و `CustomerTrailingWidgets` لتحسين قابلية القراءة والصيانة.
- [x] **الترجمة:** تم توطين النصوص التي كانت مكتوبة باللغة الإنجليزية في الكود مباشرةً باستخدام `AppLocalizations`.
